Question-1: What are the different ways to select an element in the DOM?

DOM এ একটি উপাদান নির্বাচন করার বিভিন্ন উপায় রয়েছে ঃ

১. document.getElementById("id")
শুধুমাত্র ID অনুসারে খোঁজে।

২. document.getElementsByClassName("class")
একই ক্লাসের সব এলিমেন্ট নেয়।

৩. document.getElementsByTagName("tag")
নির্দিষ্ট ট্যাগের সব এলিমেন্ট নেয়।

৪. document.querySelector("selector")
CSS সিলেক্টরের মতো কাজ করে।

৫. document.querySelectorAll("selector")
সব ম্যাচ করা এলিমেন্ট নেয়।

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হলো একটি parent এলিমেন্টে ইভেন্ট লিসেনার যুক্ত করার কৌশল, যা child এলিমেন্টের ইভেন্টগুলোর প্রতিক্রিয়া জানাতে পারে।

Question-4: What is event bubbling in the DOM?

ইভেন্ট বাবলিং হল DOM ইভেন্ট হ্যান্ডলিং-এর একটি প্রক্রিয়া, যেখানে কোনো চাইল্ড এলিমেন্টে ইভেন্ট ঘটলে, সেটি ধাপে ধাপে তার প্যারেন্ট এবং আরও উপরের এলিমেন্টগুলোতে পৌঁছে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

১.জাভাস্ক্রিপ্টে নতুন HTML এলিমেন্ট তৈরি করতে document.createElement("tagName") ব্যবহার করা হয়।

২. ওয়েবপেজে এলিমেন্ট যোগ করা
* appendChild() শেষে যোগ করা
* prepend() শুরুতে যোগ করা
* insertBefore() নির্দিষ্ট এলিমেন্টের আগে যোগ করা

৩.ওয়েবপেজ থেকে এলিমেন্ট মুছে ফেলা

* removeChild() নির্দিষ্ট এলিমেন্ট মুছতে
* replaceChild() একটি এলিমেন্টকে নতুন এলিমেন্ট দিয়ে পরিবর্তন করা
